# DUAL

어떤 구간의 통신이 불가능해지는 경우, `DUAL`은 다음과 같이 작동한다.
1. 장애가 발생한 네트워크의 라우터가 정보 공유
2. 정보를 받은 라우터는 해당 메시지에 대한 `ACK` 전송
3. 장애가 발생한 네트워크의 라우터는 이에 대하여 `EIGRP Query` 전송
4. `Query` 수신 후 `ACK`, 이후 `EIGRP Reply`를 전송
5. `Reply` 수신 후 `ACK`

```
	     장애가 발생한 구간에 
	      도달 가능한지 여부
EIGRP Query <------> EIGRP Reply
```
